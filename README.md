# ☕ iosApp1 – Coffee Run App (Tim Hortons Theme)

## 📱 Overview
This is a SwiftUI iOS app that simulates placing and saving coffee orders with a Tim Hortons theme.  
It includes three main tabs:
- **Order** – Create a new drink order (type, size, add-ons).
- **Saved Orders** – View and repeat your previous orders.
- **Timer** – Coffee run countdown timer.

## ✨ Features
- Add drink type (Coffee, Tea, Latte).
- Select drink size (Small, Medium, Large).
- Choose add-ons (Black, Cream, Milk, Sugar).
- Save orders and view them later.
- Repeat last order with one tap.
- Coffee run timer with reset option.
- Tim Hortons logo branding across all screens.

## 🛠️ Built With
- Swift
- SwiftUI
- Xcode
